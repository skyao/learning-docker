Docker基础命令:

| 命令    | 命令描述                           |
| ------- | ---------------------------------- |
| info | 显示 Docker 详细的系统信息 |
| version | 显示docker客户端和服务端版本信息 |
| inspect | 查看容器或镜像的配置信息, 默认为json数据 |
| events | 实时打印服务端执行的事件 |
